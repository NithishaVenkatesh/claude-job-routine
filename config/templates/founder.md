# Template intent — founder / CTO (early-stage startups)

Goal: founder-to-builder tone. Show I get the problem and can own real surface area fast.

Beats:
- Hook: something specific about what they're building or a recent, TRUE signal (from JD/research).
- Fit: the most impressive, relevant project + outcome; emphasize ownership and shipping.
- Value: one sentence on where I could help immediately (grounded in the role).
- Ask: "Worth a quick chat?" — direct, low-friction.
- Sign-off: name + GitHub + LinkedIn.
