# Template intent — hiring manager / engineering manager
# This is a SKELETON of intent, not fill-in-the-blanks. The model writes an original email.

Goal: show I understand what their team is building and that my agentic/RAG/backend work is
directly relevant, then ask a low-friction question.

Beats:
- Hook: one true, specific detail about the team/product/role (from the JD or research).
- Fit: one relevant project + real outcome that maps to what this role needs.
- Ask: "Are you the right person to talk to about the {role}, or could you point me to them?"
- Sign-off: name + GitHub + LinkedIn.
