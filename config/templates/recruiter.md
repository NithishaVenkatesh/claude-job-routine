# Template intent — recruiter / talent acquisition

Goal: make it easy for a recruiter to slot me to the right role. Be crisp and skimmable.

Beats:
- Hook: name the specific role + one reason the company is interesting (true detail).
- Fit: 2-3 skill/keyword matches to the JD (ATS-friendly) + one project proof point.
- Logistics: location/remote + availability, briefly.
- Ask: "Is the {role} still open, and are you the right contact for it?"
- Sign-off: name + GitHub + LinkedIn.
